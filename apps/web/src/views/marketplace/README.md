# Marketplace Views

Marketplace UI for theme/style presets and plugin discovery/management.

## Files

| File | Purpose |
|------|---------|
| `MarketplaceSidebar.svelte` | Shared marketplace shell with section tabs |
| `MarketplaceColors.svelte` | Theme preset and accent controls |
| `MarketplaceTypography.svelte` | Typography options for editor appearance |
| `MarketplacePlugins.svelte` | Plugin registry browsing and install/uninstall flows |
| `MarketplaceBundles.svelte` | Placeholder bundles view |
| `PluginMarketplace.svelte` | Full-screen marketplace implementation (legacy/alternate surface) |
