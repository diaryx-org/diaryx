# Diaryx Sync Server

A custom Y.js Websocket server to enable live sync for the Diaryx web (or Tauri) apps.

## Development

```bash
# Install dependencies
bun install

# Start development server
bun run dev
```

Server runs on `ws://localhost:1234` by default.
